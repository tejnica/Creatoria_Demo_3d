# Creatoria Demo 3D

🧠 **Creatoria** — демоверсия интерфейса AI-сервиса для многокритериального проектирования и анализа инженерных решений.  
Позволяет загружать описание задачи в YAML, запускать оптимизацию и визуализировать Pareto-фронт в 3D с пояснениями.

---

## ⚙️ Технологии

- React + TailwindCSS  
- Plotly 3D Graph  
- YAML upload + parsing  
- Логотип и стили Creatoria  

---

npm install
npm start
